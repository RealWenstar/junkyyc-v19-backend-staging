# AI JunkYYC v1.9 Backend Staging

Sanitized staging deploy repository for the v1.9 Node/Express backend.

- Render service name: `junkyyc-api-v19-staging`
- Root directory: `backend`
- Build command: `npm ci`
- Start command: `npm start`
- Required staging env: `NODE_ENV=production`, `OPENAI_API_KEY`
- Intentionally omitted on staging: Telegram and SMTP env vars
